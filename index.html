В чём преимущество VKBuddhist.js перед другими инструментами?

  1. Минимализм (~300 символов, учитывая перенос каретки и пробелы)

  2. Никаких сторонних библиотек

  3. Полная прозрачность действий, в отличии от "комбайнов", которые неизвестно что делают с вашими личными данными

  4. Моментальное удаление при любой скорости соединения

  5. Версия скрипта с возможностью исключения выбранных диалогов из удаления

//VKBuddhist.js Author: Elnee aka Daemons
var dials = document.getElementsByClassName('dialogs_del');
for (var i = 0; i < dials.length; i++) {
	dials[i].click();
}
setTimeout(function(){
	var buttons = document.querySelectorAll('button.flat_button');
	for (var i = 0; i < buttons.length; i++) {
		buttons[i].click();
	}
	location.reload();
},1000);

Инструкция:

  1. ВНИМАНИЕ! Скрипт корректно работает только в браузере Chrome

  2. Нажмите на код выше, чтобы скопировать скрипт

  3. Перейдите по ссылке vk.com/im

  4. Нажмите F12 и вставьте скрипт в консоль разработчика. Или очистите адресную строку, введите javascript: и после двоеточия вставьте в неё скрипт

  5. Нажмите Enter

Если вы хотите исключить из удаления некоторые диалоги:

  1. В поле ниже введите id тех страниц, диалоги с которыми надо оставить

  2. Вводить нужно только цифры, через запятую, без пробелов

  3. Пример: 12341234,12341234,12341234
